## Build from Source

### Build Requirements

- Rust toolchain: 1.58.0 or later
- CMake: 3.12 or later

### From crates.io

```Shell
cargo install zuk
```

### Git Repository

```Shell
git clone https://github.com/yozuk/yozuk.git
cd yozuk/zuk
cargo install .
```
