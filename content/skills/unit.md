Unit Converter

## Convert Length

```Shell
# Length
$ zuk 2000ft

# Weight
$ zuk 100lb

# Storage Size
$ zuk 20.5MiB
```

| Weight | `g` `ng` `µg` `mg` `kg` `oz.` `lb` |
| Pressure | `Pa` `hPa` `bar` `atm` `mmHg` |
| Energy | `J` `kJ` `Cal` `kCal` |
| Frequency | `Hz` `kHz` `MHz` `GHz` `THz` |
| Temperature | `°C` `°F` `K` |
| Speed | `km/h` `m/s` `mph` `kn.` |
| Storage Size | `B` `kB` `MB` `GB` `TB` `KiB` `MiB` `GiB` `TiB` |
